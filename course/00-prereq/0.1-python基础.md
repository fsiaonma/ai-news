## 章节正文

### 第 1 步：变量与结构

```python
name = "LLM"
nums = [1, 2, 3]
user = {"id": 1, "role": "admin"}

for n in nums:
    if n > 1:
        print(n)

for k, v in user.items():
    print(k, v)
```

### 第 2 步：函数与类

```python
def greet(name, lang="zh"):
    return f"Hello, {name}"

class ChatClient:
    def __init__(self, api_key: str):
        self.api_key = api_key

    def send(self, message: str) -> str:
        return message
```

### 第 3 步：文件与异常

```python
try:
    with open("config.json") as f:
        data = f.read()
except FileNotFoundError:
    data = "{}"
```
