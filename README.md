# AI 资讯日报

**在线站点：** https://z-team-z.github.io/ai-news/

每日 AI 模型、产品、论文与行业动态 — 联网检索 · 来源核验 · 按日归档。含 Agent 应用工程系统教程。

如果对你有帮助，欢迎 [关注 @zteam](https://github.com/Z-TEAM-Z) 并在 GitHub [Star ⭐ 本项目](https://github.com/Z-TEAM-Z/ai-news)，方便追踪后续资讯与教程更新。

```bash
npm run dev
```
