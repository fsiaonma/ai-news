- **日期：**The Information / Axios / The Verge 报道 2026-06-25；Altman 员工备忘录同日流出。

- **核验状态：**已用 Axios、The Verge 原文交叉核验；OpenAI 未发独立公告；GPT-5.6 具体 benchmark 与发布日期未确认。

## 论文与研究

### 1. Murakkab：云端 agent 工作流资源高效编排

- **日期：**MIT News 2026-06-25；论文将发表于 OSDI 2026（7 月 13–15，Seattle）。
- **主题：**MIT CSAIL 与 Microsoft Azure 提出 **Murakkab**，以 declarative abstraction 解耦 agent 工作流规格与执行配置；profile-guided optimizer + adaptive runtime 联合管理模型、工具与硬件映射，动态满足 accuracy / latency / energy / cost SLO。
- **结果：**在 video Q&A、code generation 等工作流上，GPU 用量降最多 2.8×、能耗 3.7×、成本 4.3×；一项配置在 accuracy 仅降约 2% 情况下能耗降一个数量级以上。
- **为什么重要：**agent 工作流碎片化导致 cloud 过度分配资源；与 Accenture token rationing、Murakkab 类系统优化形成「agent 成本治理」主线。
- **核验状态：**已用 MIT News、Microsoft Research 出版物页、arXiv 摘要核验。
- **来源：**[MIT News](https://news.mit.edu/2026/improving-ai-agent-speed-and-energy-efficiency-0625)，[Microsoft Research](https://www.microsoft.com/en-us/research/publication/murakkab-resource-efficient-agentic-workflow-orchestration-in-cloud-platforms/)，[arXiv:2508.18298](https://arxiv.org/abs/2508.18298)

## 行业动态

### 1. AutoJack：恶意网页劫持 AutoGen browsing agent

- **日期：**漏洞披露 2026-06-24；adyog 分析 2026-06-26。
- **事件：**Microsoft **AutoGen Studio** 存在漏洞链：browsing agent 以 localhost 身份运行 → MCP WebSocket 对 localhost 跳过认证 → 认证后可 spawn 任意进程。恶意网页可 instruct agent 连接本地 MCP 并执行任意命令，实现「drive-by hijack」。
- **为什么重要：**agent 浏览不可信网页时的安全边界不在网络 perimeter，而在每个 URL / API / 文档；与 GPT-5.6 政府安全审查、Anthropic Mythos 网络安全能力争议同属 agent 风险治理。
- **核验状态：**已用 adyog 分析文核验架构描述；Microsoft 补丁状态本轮未查官方 CVE 页。
- **来源：**[adyog - AutoJack](https://pulse.adyog.com/insights/autojack-one-web-page-hijacks-microsoft-autogen-ai-agent)

### 2. Anthropic 指控阿里蒸馏 Claude：Ars 深度跟进与中方回应信号

- **日期：**Anthropic 信函 2026-06-10；CNBC 等 6-24 报道；Ars Technica 2026-06 跟进。
- **事件：**Anthropic 致美国参议员，称阿里 Qwen 实验室 2026-04-22 至 06-05 期间用约 2.5 万虚假账号、2880 万次交互实施迄今最大 Claude 蒸馏攻击，且发生在 Trump 4 月反蒸馏备忘录之后。Anthropic 建议更新反垄断信息共享、加强芯片出口管制、惩罚蒸馏行为。阿里未回应媒体；阿里周二另起诉 Trump 政府，指控错误将其与军方关联并列入黑名单（Reuters 报道）。360 创始人周鸿祎据 SCMP 在北京网络安全大会称 Mythos 如「网络核武器」，中国须自建同类能力——为迄今 prominent 中国科技创始人公开回应 Mythos 战略风险的第一例。
- **核验状态：**Anthropic 指控已用 Ars 原文核验；阿里未确认或否认；周鸿祎言论为二手媒体报道，非官方政策。
